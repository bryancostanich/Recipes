# Chipotle Baja Sauce

## Ingredients

* (1/3 Cup) Vegenaise
* (2-3 large cloves) Garlic, finely minced
* (2 Tbsp) [Tia Lupita Chipotle](https://tialupitafoods.com/products/chipotle-hot-sauce)
* (1/2 Tsp) Salt
* (2) Lime's Juice

## Instructs

1. Mix together.
2. Enjoy in [Baja Fish Tacos](../../Mains/Baja_Fish_Tacos/readme.md)
